# Description
Simple example of opencv use in C++.

# How to run

#### Opencv installation:
```bash
sudo apt install libopencv-dev
```

#### Build and run:
```bash
make           # Builds the project
./main         # Runs the executable
```

#### Clean:
```bash
make clean     # Cleans up build files
```
